# Station Images

Place station logo images here. They will be served at `/static/images/stations/<filename>`.

## Expected files for default stations:

- `nova937.png` - Nova 93.7
- `mix945.png` - Mix 94.5
- `96fm.png` - 96FM
- `triplem.png` - Triple M Perth
- `triplej.png` - Triple J

## Recommended format:
- PNG or JPG
- Square aspect ratio (e.g., 300x300)
- Transparent background for PNG (optional)
